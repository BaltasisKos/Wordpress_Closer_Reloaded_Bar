# 🍸 Closer Reloaded Bar Website (WordPress Project)

Welcome to the repository for the **Closer Reloaded Bar** website. This project is a fully dynamic WordPress website developed to showcase the bar's unique atmosphere, menu, upcoming events, and location. 

I created this project to demonstrate my skills in **WordPress development, custom staging/configuration, and frontend presentation**.

---

## 🎯 Project Overview

The main goal of this project was to build a modern, responsive, and easily manageable website for a nightlife venue. It serves as a digital hub where users can check the menu, book reservations, or see event schedules.

### Key Features:
* **Dynamic Menu Display:** Easily updatable food and drinks menu.
* **Event Management:** A clean layout showcasing upcoming DJ sets, parties, and special nights.
* **Fully Responsive:** Optimized for seamless browsing on mobile devices (crucial for customers on the go).
* **SEO & Performance Optimized:** Built with best practices to ensure fast loading times.
